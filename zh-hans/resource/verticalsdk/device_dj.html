ITuyaVerticalSdk centralControlSdk = TuyaOptimusSdk.getManager(ITuyaVerticalSdk.class);

1.2. ITuyaLightDevice 接口介绍

1.2.1. 获取当前灯具备的功能（几路灯）

LightType getLightType();

参数说明

返回值 说明
LightType 获取当前是几路灯（是依据灯支持的照明功能分类）并且根据等选择不同的支持模式
灯类型枚举值 说明
TYPE_C 白光灯（一路灯），dpCode：bright_value
支持的工作模式为 LightMode#MODE_WHITE 白光模式
TYPE_CW 白光+冷暖（二路灯），dpCode：bright_value + temp_value
支持的工作模式为 LightMode#MODE_WHITE 白光模式
TYPE_RGB RGB（三路灯），dpCode：colour_data
支持的工作模式为 LightMode#MODE_COLOUR 彩光模式
LightMode#MODE_SCENE 情景模式
TYPE_RGBC 白光+RGB（四路灯），dpCode：bright_value + colour_data
支持的工作模式为 全都支持：LightMode#MODE_WHITE 白光模式； LightMode#MODE_COLOUR 彩光模式 ；LightMode#MODE_SCENE 情景模式
TYPE_RGBCW 白光+冷暖+RGB（五路灯），dpCode：bright_value + temp_value + colour_data
支持的工作模式为 全都支持：LightMode#MODE_WHITE 白光模式； LightMode#MODE_COLOUR 彩光模式 ； LightMode#MODE_SCENE 情景模式

1.2.2. 获取灯LightDataPoint数据

LightDataPoint getLightDataPoint();

参数说明

返回值 说明
LightDataPoint 包含
boolean powerSwitch 开关状态
LightMode workMode 工作模式 枚举
int brightness 亮度百分比值 从1～100 步长1
int colorTemperature 色温百分比值 从1～100 步长1
LightColourData colorHSV 颜色值
LightScene scene 彩灯的情景模式

补充参数说明

数据类型 说明
LightMode 枚举类型 包含
MODE_WHITE("white") 白光模式
MODE_COLOUR("colour") 彩光模式
MODE_SCENE("scene") 情景模式
LightColourData 包含HSV
h ：色调，hue，取值范围0-360
s ：饱和度，saturation，取值范围0-100
v ：明度，value，取值范围0-100
LightScene 枚举类型 情景模式 包含
SCENE_GOODNIGHT 晚安情景
SCENE_WORK 工作情景；
SCENE_READ 阅读情景；
SCENE_CASUAL 休闲情景；

1.2.3. 获取当前灯支持的工作模式

List<LightMode> getSupportWorkMode();

参数说明

返回值 说明
List<LightMode> 依据当前灯的类型（属于几路灯）返回支持的工作模式集合

1.2.4. 获取所支持的场景集合

List<LightScene> getSupportLightScene();

参数说明

返回值 说明
List<LightScene> 返回固定的包含
SCENE_GOODNIGHT 晚安情景
SCENE_WORK 工作情景；
SCENE_READ 阅读情景；
SCENE_CASUAL 休闲情景四种场景的的集合

IControlLight operate();

参数说明

返回值 说明 参数 说明

1.3.5. 控制灯的颜色值

IControlLight colorHSV(int hue, int saturation, int value);

参数说明

返回值 说明 参数 说明
IControlLight 返回当前实例，链式调用 int hue,
int saturation,
int value		 hue 色调 （范围：0-360）
saturation 饱和度（范围：0-100）
value 明度（范围：0-100）

示例代码

//控制灯的颜色值
lightDevice.operate()
    .colorHSV(20,200,200).publish(new IResultCallback() {
    @Override
    public void onError(String code, String error) {
        System.out.println("code = " + code + ", error = " + error);
    }

    @Override
    public void onSuccess() {

    }
});
