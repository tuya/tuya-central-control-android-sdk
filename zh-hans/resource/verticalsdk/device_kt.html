参数说明

返回值 说明

1.1.2. 获取空调的温度控制范围步长

StandardStepRangeBean getTempRangBean();

参数说明

返回值 说明
StandardStepRangeBean 返回包含
String unit; 单位
int min; 最小范围
int max; 最大范围
int step; 变化的步长
的信息 当max=min=0则数据获取失败

1.1.3. 获取摆风模式

SwingType getCurrentSwingType();

参数说明

返回值 说明
SwingType 返回当前空调摆风模式
SWING_HORIZONTAL = 0;//水平
SWING_VERTICAL = 1;//垂直
SWING_NONE = 2; //不支持
SWING_BOTH = 3;// 都支持
boolean horizontalEnabled
boolean verticalEnabled;

1.1.4. 获取当前设置的工作模式

FanMode getCurrentFanMode();

参数说明

返回值 说明
FanMode 枚举类型 返回空调当前设置的工作模式
目前提供的主要有： AUTO("auto"), COLD("cold"), HOT("hot"), WET("wet"), WIND("wind"), ECO("eco"), FLOOR_HEAT("floor_heat"), FLOOR_HEAT_AND_HEAT("floor_heat_and_heat"), UNKNOWN("unknow"); 模式

//设置空调温度为20度
airConditionDevice.operate()

    @Override
    public void onError(String code, String error) {
        System.out.println("code = " + code + ", error = " + error);
    }

    @Override
    public void onSuccess() {
    }
});
